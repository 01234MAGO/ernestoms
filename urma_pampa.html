TRADUCCIÓN DE DATOS HISTÓRICOS SOBRE EL AGRO

SOBRE ESTE PROYECTO

En colaboración con el laboratorio URMA PAMPA, este proyecto marca mi primera experiencia formal como científico de datos. El objetivo principal es utilizar la información histórica sobre las muestras del laboratorio para que la ingeniera pueda brindar conferencias informativas. Además, nos enfocamos en maximizar el valor de estos datos.
Agradezco la confianza depositada en mí al brindarme acceso a sus datos y manejo la información con la máxima responsabilidad en cuanto a confidencialidad.

FUNCION DE EXTRACCION

Para extraer y organizar datos de archivos Excel, desarrollé una función que devuelve los datos de cada celda según sus coordenadas, facilitando su manejo y análisis. También implementé una función adicional para descomprimir archivos ZIP que contienen estos archivos Excel. Utilicé las bibliotecas openpyxl, csv, io y zipfile para gestionar eficazmente la manipulación de datos y archivos comprimidos.

CODIGO DE LA FUNCION

INFORMACION VALIOSA

Siguiendo la visión de URMA PAMPA, los datos se convierten en información valiosa que facilita a los productores tomar decisiones fundamentadas. Compartir esta información con los clientes es clave para asegurar que cuenten con los insights necesarios para optimizar sus estrategias y operaciones.

HABILIDADES INVOLUCRADAS

INGENIERIA DE DATOS

Los datos se registraban inicialmente en cuadernos escritos y, actualmente, se emiten certificados y se guardan en formato Excel. Mi objetivo fue unificar estos formatos. Para extraer los datos en Excel, desarrollé una función de recolección. Además, creé una planilla de carga manual para los datos escritos, asegurando que todo se almacenara en el mismo formato.
Una vez unificados, los datos se almacenaron en una base de datos para su gestión eficiente.

ANALISIS DE DATOS

En esta etapa, fue esencial la colaboración con los profesionales del laboratorio. Ellos guiaron el análisis, definiendo las variables y características de su interés. Mi tarea consistió en identificar las medidas estadísticas relevantes y explicar su significado.
Este proceso fue un enriquecedor aprendizaje y descubrimiento conjunto.

INFORMES Y PRESENTACIONES

Parte de mi trabajo consistió en crear tableros interactivos para generar informes personalizados destinados a productores agrícolas. Además, preparé presentaciones tipo PowerPoint para charlas informativas y capacitaciones, proporcionando así soporte fundamental para la educación y la toma de decisiones.
