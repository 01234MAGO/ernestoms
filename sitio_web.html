DISEÑO DE PORTAFOLIO WEB

SOBRE ESTE PROYECTO

La idea surge de construir un espacio para mostrar mis habilidades, logros y proyectos, que además refleje la dedicación y creatividad involucrada en cada cosa que hago. Por eso consideré armar un sitio web, lo que representó un desafío dado que no tenía conocimientos previos en desarrollo web.
La asistencia de ChatGPT fue fundamental en la capacitación técnica y me guió en la resolución de problemas y funcionalidad del sitio. Debo aclarar que el proyecto está montado sobre una plantilla como punto de partida.

SUPERVISION HUMANA Y ARTIFICIAL

El desarrollo de este proyecto contó con la asistencia de ChatGPT y la supervisión de Federico Vignolo, un amigo que está iniciando su carrera en el desarrollo web.
Una etapa desafiante fue crear prompts para recibir consejos y soluciones rápidas sobre HTML, CSS. Mientras que mi compañero aportó su experiencia práctica para optimizar el diseño.
Esta colaboración entre la inteligencia artificial y el apoyo humano no solo hizo que el proceso fuera más eficiente, sino también enriquecedor, destacando cómo la combinación de tecnología y trabajo en equipo puede mejorar la creatividad y productividad en proyectos complejos.

HABILIDADES INVOLUCRADAS

LENGUAJES HTML Y CSS

A través de este proyecto, aprendí los fundamentos de HTML y CSS, utilizando HTML para estructurar contenido con encabezados, párrafos y enlaces, y CSS para dar estilo y formato a la página. Apliqué etiquetas semánticas para mejorar accesibilidad y SEO, y utilicé técnicas de diseño responsivo para asegurar una buena apariencia en diversos dispositivos. Este proyecto amplió mis conocimientos técnicos y mejoró mi capacidad para resolver problemas en el desarrollo web.

DISEÑO

El diseño de este proyecto se centró en la coherencia visual y la estética profesional, usando una paleta de colores neutros y elegantes, combinando tonos de azul para transmitir confianza y seriedad. La tipografía moderna y legible facilita la lectura en todos los dispositivos. Apliqué CSS para mantener colores y fuentes uniformes, ajustando márgenes y espaciados para un diseño limpio y equilibrado. Elementos visuales, como iconos y botones, fueron integrados para mejorar la navegación y la experiencia del usuario, asegurando que cada componente contribuyera a la funcionalidad y apariencia del sitio.
